# Dealer's Choice

A vanilla style expansion for Balatro.

Requires [Steamodded](https://github.com/Steamodded/smods/releases/latest)

<img width="160" height="164" alt="dealersChoice_logo" src="https://github.com/user-attachments/assets/401ffbb3-b3f5-485c-84a1-f4ece3e5c25d" />

Adds 19 Jokers, 1 Spectral Card, and 1 Seal as of version 1.0.0.

You can find full details [here.](https://docs.google.com/document/d/1hKXrfDhfFaUq5a1z63INJxVhS0jty3_npE-oV7sSuro/)
